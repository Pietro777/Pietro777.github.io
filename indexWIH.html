HTML is a language used to describe a web page.
·HTML is Hyper Text Markup Language.
·HTML is not a programming language,but a markup language.
·The markup language is a set of tag tags.
·HTML use tag tags to describe web pages.


HTML markup tags are often referred to as HTML tags.
·HTML tags are keywords surrounded by Angle brackets.
·HTML tags usually come in pairs.
·The first tag is the start tag, and the second tag is the end tag.
·The start and end tags are also known as open labels and closed tags.


HTML documents = webpages
·HTML documents descript webpages.
·HTML documents contain HTML tags and plain text.
·HTML documents are also called webpages.
