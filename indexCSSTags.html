CSS rules consist of two main parts: the selector, and one or more 
statements.

selector {declaration1; declaration2; ... declarationN }

The selector is usually the HTML element that you need to change the 
style.
Each declaration consists of a property and a value.
Property is the style attribute that you want to set. Each attribute 
has a value. Properties and values are separated by colons.

selector {property: value}

The following line of code defines the text color in the h1 element as
 red, and the font size is set to 14 pixels.
In this example, h1 is the selector, color and font-size are attributes,
 red and 14px are values.

 h1 {color:red; font-size:14px;}

 Most stylesheets contain more than one rule, and most rules contain 
 more than one statement. The use of multiple declarations and Spaces 
 makes the style sheet easier to edit:

body {
  color: #000;
  background: #fff;
  margin: 0;
  padding: 0;
  font-family: Georgia, Palatino, serif;
  }

  Whether to include Spaces does not affect CSS work in the browser, 
  and again, unlike XHTML, CSS is not case sensitive. There is one 
  exception though: class and id names are sensitive to case if they 
  involve working with HTML documents.
